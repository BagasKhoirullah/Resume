Konsep Dasar

Constructor

Jenis Constructor

  • Default Constructor
  • Named Constructor
  • Parameterized Constructor
  • Constant Constructor
  • Factory Constructor
Example

Named Constructor

class Mobil { String? merk; int? tahun; Mobil(this.merk,this.tahun); Mobil.baru(){ merk='Toyota'; tahun=2023; } Mobil.dariJson(Map json){ merk=json['merk']; tahun=json['tahun']; } }
Constant

Constant Constructor

  • Digunakan untuk objek immutable
  • Semua field harus final
  • Menghemat memory
class Warna { final int r,g,b; const Warna(this.r,this.g,this.b); }
Encapsulation

Pengantar Enkapsulasi

  • Menyembunyikan detail implementasi
  • Melindungi data
  • Mengontrol akses data
  • Menggunakan underscore (_) untuk private
Getter

Mengakses Data

class Person{ String _name; String get nama{ return _name; } }
Setter

Memodifikasi Data

set nama(String value){ if(value.length >= 3){ _name = value; } }

Getter & Setter Lengkap

class Produk{ String _nama; double _harga; String get nama => _nama; double get harga => _harga; set harga(double value){ if(value > 0){ _harga = value; } } }
