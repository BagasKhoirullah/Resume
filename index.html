Named Constructor

class Mobil {

String? merk;
int? tahun;

Mobil(this.merk,this.tahun);

Mobil.baru(){
merk='Toyota';
tahun=2023;
}

Mobil.dariJson(Map json){
merk=json['merk'];
tahun=json['tahun'];
}

}

Constant Constructor

class Warna{

final int r,g,b;

const Warna(this.r,this.g,this.b);

}

Enkapsulasi

Enkapsulasi adalah teknik menyembunyikan data dengan menggunakan field private.

  • Melindungi data
  • Mengontrol akses
  • Menggunakan underscore (_) untuk private

Studi Kasus Login

class User{

String _username;
String _password;

User(this._username,this._password);

bool login(String u,String p){
return _username==u && _password==p;
}

}

Latihan
